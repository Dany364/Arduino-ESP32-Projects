# 🌐 Controlul unui LED prin intermediul unei pagini web utilizand ESP32

![Schema](Schema.png)

---

# 📖 Descriere

Acest proiect demonstreaza utilizarea modulului Wi-Fi integrat al placii **ESP32** pentru realizarea unei interfete web de control.

ESP32 se conecteaza la o retea Wi-Fi si functioneaza ca un server web local. Prin accesarea adresei IP a dispozitivului dintr-un browser, utilizatorul poate controla starea unui LED prin intermediul unei interfete grafice simple.

Proiectul evidentiaza modul de utilizare al conectivitatii Wi-Fi oferite de ESP32 si reprezinta o baza pentru dezvoltarea aplicatiilor de tip Internet of Things (IoT).

---

# 🔧 Componente utilizate

- ESP32
- LED
- Rezistenta 220 ohmi
- Breadboard
- Fire de conexiune
- Retea Wi-Fi
