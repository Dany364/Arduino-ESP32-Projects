# 🚀 Portofoliu de proiecte Arduino si ESP32

Bine ai venit!

Acest repository contine o colectie de proiecte dezvoltate utilizand placile **Arduino Mega 2560** si **ESP32**, realizate in cadrul activitatilor universitare si al studiului individual.

Scopul acestui repository este prezentarea cunostintelor dobandite in domeniul sistemelor embedded, al comunicatiilor wireless si al interfatarii senzorilor si actuatoarelor.

---

# 🛠️ Tehnologii utilizate

- Arduino Mega 2560
- ESP32
- C/C++
- Python
- Bluetooth
- ESP-NOW
- Wi-Fi
- RFID
- EEPROM
- LCD
- OLED
- LED Bar
- Relee
- Motoare Stepper
- Senzori analogici si digitali

# 🎯 Obiectiv

Acest repository reprezinta portofoliul meu de proiecte din domeniul sistemelor embedded.

Proiectele acopera o gama variata de aplicatii, precum:

- comunicatii wireless (Bluetooth, WiFi, ESP-NOW);
- sisteme RFID;
- senzori analogici si digitali;
- actuatoare;
- afisaje LCD, OLED si LED;
- automatizari;
- jocuri interactive;
- interfatarea componentelor hardware cu microcontrolere Arduino Mega si ESP32.
